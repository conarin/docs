当サービスでは、以下の情報を収集しています。
なお、サービスによっては本プライバシーポリシーの内容に加え、個別に収集する内容が加わる場合があります。その場合は、該当サービスのドキュメントページに収集する情報を記載しておりますのでご確認ください。

また、本プライバシーポリシーでの「当サービス」や「サービス提供者」などの言葉は[利用規約](/tos)で定義されているものと同じものを指します。

## 当サービスが収集する情報
### Webサイト
当サービスのうち、Webで提供するサービス(以下、「当Webサイト」といいます。)を利用する場合には、以下の情報を収集します。
1. 利用者のデバイスに関する情報  
    当Webサイトでは、利用者のデバイスに関する情報をアクセスログという形で記録しています。これには、利用者のIPアドレス、OS、ブラウザなどの情報を含みます。
2. 当Webサイトの利用に関する情報  
    当Webサイトでは、当Webサイトの利用に関する情報をアクセスログという形で記録しています。これには、利用者がいつどのページにアクセスしたかなどの情報を含みます。
3. Discordアカウントの情報（一部サービス）  
    当Webサイトのうち、Discordアカウントで認可することで利用できるサービスでは、DiscordアカウントのIDやユーザー名など、利用者が認可した情報を取得します。

### Discord Bot
当サービスのうち、コミュニケーションプラットフォーム「Discord」で提供するBotアプリケーション（以下、「当アプリケーション」といいます。）を利用する場合には、以下の場合において情報を収集します。
1. 当アプリケーションを利用するうえで必要な場合  
    利用するアプリケーションによって異なります。該当アプリケーションのドキュメントページをご確認ください。
2. 当アプリケーションの利用において例外またはエラーが発生した場合
    1. 利用者のユーザーおよびメンバー情報
    2. 入力したサーバーの情報
    3. 入力したチャンネルの情報
    4. 入力したコマンドの内容
3. 当アプリケーションが参加または退出した場合
    1. 参加または退出したサーバーの情報
4. その他、サービス提供者が当アプリケーションの改善や開発において必要と判断した場合

## 情報の保存
収集した情報は当サービスを運用しているコンピュータ上に保存されます。これには、クラウドサービスやVPS、自宅サーバーなどを含みます。

## 情報の利用
収集した情報は以下の目的で利用します。
1. 当サービスを提供するため
2. 当サービスの維持や改善するため
3. 当サービスのバグや不具合を修正するため
4. 利用者へ連絡するため
5. 利用規約に違反した利用者の利用を制限するため
6. 当サービスの利用状況を解析するため
7. その他、上記利用目的に付随する目的

## アクセス分析
当Webサイトでは、アクセス分析に[Cloudflare Web Analytics](https://www.cloudflare.com/ja-jp/web-analytics)を利用しています。

## 情報の第三者提供
収集した情報は以下の場合を除いて、第三者に提供することはありません。
1. 本人の同意がある場合
2. 法令で認められる場合
3. 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
4. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
5. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

## 情報の訂正および削除
利用者は、当サービスが収集した情報に対して、訂正、追加または削除を請求することができます。  

## 本プライバシーポリシーの変更
本ポリシーの内容は、法令そのほか本ポリシーに別段の定めのある事項を除いて、利用者に通知することなく変更することができるものとします。  
サービス提供者が別途定める場合を除いて、変更後のプライバシーポリシーは、本プライバシーポリシーのページに掲載した時から効力を生じるものとします。
