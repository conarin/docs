<DiscordMessages>
    <DiscordMessage><DiscordMention>[非公式] 整地鯖Bot</DiscordMention> crop \<サーバー名> \<ワールド名> \<X座標1> \<Z座標1> \<X座標2> \<Z座標2> \[スキン表示]</DiscordMessage>
</DiscordMessages>
<br/>

ワールドの指定した範囲のマップ画像を表示します。

:::warning

Cloudflareによる保護のため、こちらのコマンドは現在動作しません。

:::

| パラメータ名 | 概要                  | 入力形式                                                            | 必須  | 初期値 |
|--------|---------------------|-----------------------------------------------------------------|-----|-----|
| サーバー名  | サーバー名を入力します。        | \<s1\|s2\|s3\|s5\|s7><br/><br/>左からアルカディア、エデン、ヴァルハラ、整地専用、公共施設です。 | :o: |     |
| ワールド名  | ワールド名(キーワード)を入力します。 | 第一整地<br/>整地1<br/>メイン など                                         | :o: |     |
| X座標1   | 開始地点のX座標を入力します。     | 半角のアラビア数字(整数)                                                   | :o: |     |
| Z座標1   | 開始地点のZ座標を入力します。     | 半角のアラビア数字(整数)                                                   | :o: |     |
| X座標2   | 終了地点のX座標を入力します。     | 半角のアラビア数字(整数)                                                   | :o: |     |
| Z座標2   | 終了地点のZ座標を入力します。     | 半角のアラビア数字(整数)                                                   | :o: |     |
| スキン表示  | プレイヤーのスキンを表示します。    | \[skin\|head]                                                   |     |     |

:::note

座標は小数を切り捨てた値を入力します。  
例1：1.9 → 1  
例2：-4.8 → -5  

:::
