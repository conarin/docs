    <DiscordMessage>/daily</DiscordMessage>
</DiscordMessages>
<br/>

ログインボーナスを受け取ります。ポイントと経験値がランダムな数だけ付与されます。  
サーバーごとに1日1回受け取ることができます。受け取りのリセットは日本標準時の0時です。

## 例
<DiscordMessages compactMode>
    <DiscordMessage>/daily</DiscordMessage>
</DiscordMessages>
