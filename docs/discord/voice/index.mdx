## 概要
ボイスチャンネルにて自身の音声を録音するBotです。最大30秒まで録音することができます。  
Discord Bot Portal JP でのBot案を基に作成したBotです。

## コマンド一覧
| コマンド名 (別名) | 概要                            |
|------------|-------------------------------|
| record     | 録音を開始します。                     |
| download   | 録音した音声データを送信します。              |
| play       | 録音した音声を再生します。                 |
| note       | 録音した音声のノート(メモ)を設定します。         |
| permission | 指定した音声データの再生、ダウンロードの権限を設定します。 |
| remove     | 録音した音声を録音一覧から削除します。           |
| list       | 録音した音声の一覧を表示します。              |
| join       | ボイスチャンネルに接続します。               |
| leave      | ボイスチャンネルを切断します。               |
| prefix     | プレフィックスを設定します。                |
| links      | 各種リンクを表示します。                  |
