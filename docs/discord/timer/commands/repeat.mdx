<DiscordMessages>
    <DiscordMessage><DiscordMention>JSTimerBot</DiscordMention> repeat \<日にち|曜日> \<時刻> [メッセージ] [チャンネル]</DiscordMessage>
</DiscordMessages>
<br/>

特定の曜日や日にちに通知するタイマーをセットします。  
コマンド名は`cron`、`schedule`、`interval`でも可能です。

| パラメータ名 | 概要                                                                                         | 必須  |
|--------|--------------------------------------------------------------------------------------------|-----|
| 曜日     | 通知する曜日または日にちを入力します。<br/>詳しくは[日にち/曜日の指定方法](#日にち曜日の指定方法)をご確認ください。                            | :o: |
| 日時     | 通知する時刻を入力します。<br/>詳しくは[時刻の指定方法](#時刻の指定方法)をご確認ください。                                         | :o: |
| メッセージ  | 通知時に表示させるメッセージを指定できます。255文字まで入力できます。<br/>メンションの扱いについては[メンションについて](../mention.mdx)をご確認ください。 |     |
| チャンネル  | 通知するチャンネルを指定できます。<br/>指定方法については[チャンネルの指定方法について](../channel.mdx)をご確認ください。                   |     |

## 例
<DiscordMessages compactMode>
    <DiscordMessage><DiscordMention>JSTimerBot</DiscordMention> repeat 日曜日 18:30 サザエ #通知チャンネル</DiscordMessage>
    <DiscordMessage><DiscordMention>JSTimerBot</DiscordMention> cron 毎日 0時 日付が変わりました</DiscordMessage>
    <DiscordMessage><DiscordMention>JSTimerBot</DiscordMention> interval 25日 09:00 お賃金 <DiscordMention type="channel">チャンネル名</DiscordMention> </DiscordMessage>
</DiscordMessages>

## 日にち/曜日の指定方法
毎月○日、毎日、毎週○曜日のいずれかを設定できます。

---

### 日にち
\<1~31>日 または 毎日

#### 例
- 1日
- 15日
- 毎日

---

### 曜日
\<月|火|水|木|金|土|日>曜[日]

#### 例
- 水曜日
- 土曜日
- 日曜

---

## 時刻の指定方法
時刻を入力します。時分秒まで指定でき、「時」は必須でそれ以降は任意です。  
`時` `分` `秒`か`:`(コロン)のどちらかで区切ってください。  
また、時間は24時間表記で入力してください。

#### 例
- 00:00:00
- 5:30
- 18時30分
