特定の日時に通知するタイマーをセットします。

| パラメータ名 | 概要                                                                                         | 必須  |
|--------|--------------------------------------------------------------------------------------------|-----|
| 日時     | 通知する日時を入力します。<br/>詳しくは[日時の指定方法](#日時の指定方法)をご確認ください。                                         | :o: |
| メッセージ  | 通知時に表示させるメッセージを指定できます。255文字まで入力できます。<br/>メンションの扱いについては[メンションについて](../mention.mdx)をご確認ください。 |     |
| チャンネル  | 通知するチャンネルを指定できます。<br/>指定方法については[チャンネルの指定方法について](../channel.mdx)をご確認ください。                   |     |

## 例
<DiscordMessages compactMode>
    <DiscordMessage author="user"><DiscordMention>JSTimerBot</DiscordMention> set 12月25日の午後6時</DiscordMessage>
    <DiscordMessage author="user"><DiscordMention>JSTimerBot</DiscordMention> set 1m30s 1分30秒経ちました チャンネル名</DiscordMessage>
    <DiscordMessage author="user"><DiscordMention>JSTimerBot</DiscordMention> set 2024 ことよろ <DiscordMention type="channel">チャンネル名</DiscordMention> </DiscordMessage>
</DiscordMessages>

## 日時の指定方法

---

### 自然言語
日本語で日時を入力する方法です。  
簡単な入力であれば、よしなに解釈してくれます。

#### 例
- 来週水曜日の午後8時
- 明日の26時
- 令和六年の師走の25日朝6時

---

### 西暦月日時分秒
西暦から秒までを半角のスペースで区切って入力する方法です。  
西暦より後は省略することができます。省略した月日は1、時分秒は0に自動で指定されます。  
また、西暦を省略して月から入力することもできます。今年か来年の西暦が自動で指定されます。

#### 例
- 2038 1 19 12 14 7  
→ 2038年01月19日 12時14分07秒
- 2024 1  
→ 2024年01月01日 00時00分00秒
- 1  
→ 来年の1月1日 00時00分00秒

---

### 数値+単位
現在の時刻から○○後に指定する方法です。

#### 例
- 1d2h30m
- 1日と6時間後
- 1.5分

以下の表の単位を使用できます。

| 秒      | 分      | 時    | 日   | 週    | 月     | 年    |
|--------|--------|------|-----|------|-------|------|
| s      | m      | h    | d   | w    |       | y    |
| sec    | min    |      |     |      | mo    |      |
| second | minute | hour | day | week | month | year |
|        |        | 時間   |     | 週間   | か月    |      |
|        |        |      |     |      | ヵ月    |      |
|        |        |      |     |      | ヶ月    |      |

---

:::tip

- 相対日時(`明日`や`10分後`など)で指定すると秒単位で同じ時刻に設定されます。
- 週の始まりは日曜日と定義しています。そのため、例えば1日(月)に「来週の日曜日」と入力すると7日(日)に通知されます。
- testコマンドを利用することで、日時やメッセージがどのように設定されるか確認することができます。
- セット完了メッセージのキャンセルボタンを押すと、タイマーはキャンセルされます。

:::
