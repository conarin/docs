> <Mention>@JSTimerBot</Mention> list [all] [link|url]

セットしているタイマーの一覧を表示します。  
コマンド名は`ls`でも可能です。

| パラメータ名      | 概要                               | 必須 |
|-------------|----------------------------------|----|
| all         | コマンドを実行したサーバー以外でセットしたタイマーも表示します。 |    |
| link \| url | セット時のメッセージへのリンクを表示します。           |    |

## 例
- <Mention>@JSTimerBot</Mention> list
- <Mention>@JSTimerBot</Mention> list url
- <Mention>@JSTimerBot</Mention> ls all link
