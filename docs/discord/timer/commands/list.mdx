---
description: セットしているタイマーの一覧を表示します。
---

<DiscordMessages>
    <DiscordMessage><DiscordMention>JSTimerBot</DiscordMention> list [all] [link|url]</DiscordMessage>
</DiscordMessages>
<br/>

セットしているタイマーの一覧を表示します。  
コマンド名は`ls`でも可能です。

| パラメータ名      | 概要                               | 必須 |
|-------------|----------------------------------|----|
| all         | コマンドを実行したサーバー以外でセットしたタイマーも表示します。 |    |
| link \| url | セット時のメッセージへのリンクを表示します。           |    |

## 例
<DiscordMessages compactMode>
    <DiscordMessage><DiscordMention>JSTimerBot</DiscordMention> list</DiscordMessage>
    <DiscordMessage><DiscordMention>JSTimerBot</DiscordMention> list url</DiscordMessage>
    <DiscordMessage><DiscordMention>JSTimerBot</DiscordMention> ls all link</DiscordMessage>
</DiscordMessages>
